HideButtons plugin
==================

Hide buttons from the Galicaster non-admin UI

Settings
--------

+---------+------+-----------------------------------------------------------+
| Setting | Type   | Description                                             |
+=========+========+=========================================================+
| hide    | string | A space-separated list of buttons to be hidden from the |
|         |        | non-admin UI, possible values are: 'help', 'pause',     |
|         |        | 'record', 'stop', 'swap'                                |
|         |        |                                                         |
|         |        | e.g. ::                                                 |
|         |        |                                                         |
|         |        |   [hidebuttons]                                         |
|         |        |   hide = record help                                    |
|         |        |                                                         |
+---------+--------+---------------------------------------------------------+
